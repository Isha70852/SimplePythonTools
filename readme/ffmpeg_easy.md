使用python語言，使用flet為gui，調用ffmpeg的簡易轉檔/縮小影片檔案程式。

功能:

1. 單檔/批次縮小影片檔案 (透過減小bitrate達成)
2. 單檔/批次轉換影片/音樂檔案

  
使用:  
方法1:
1. 安裝ffmpeg並將其加入path中。
2. 下載exe檔案並運行

方法2:
1. 安裝python
2. 安裝運行所需套件
3. 安裝ffmpeg並將其加入path中。
4. 下載程式碼並運行
